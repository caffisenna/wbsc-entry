## System Requirements
1. PHP ver8.2 or later
2. [composer]('https://getcomposer.org/)
3. RDBMS (MySQL8 or later)


## Installation
1. `git clone git@github.com:caffisenna/wbsc-entry.git`
2. run `composer install`
